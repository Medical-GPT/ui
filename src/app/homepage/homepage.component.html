Welcome to the Medical Chatbot

Description This is a medical chatbot application where you should provide information about your symptoms and the AI will give you advice on what you should do and how you should take care of it along with suggestions on which doctors you should go. The sidebar consists of 4 models each showing the 4 stages of the project. Steps Pretraining

1. Pretraining

The model is shown examples of the English language and tries to learn speaking (explain pretraining for GPT models).

Clean GPT2

2. GPT2

Loaded pretrained GPT2 model. It shows how a pretrained model behaves - it tries to complete the provided text rather than give concrete answers.

Medical GPT2

3. Fine-tuned GPT2 model for medical Q/A

Here the model has been fine-tuned on thousands of examples in order to produce answers to medical questions.

Empathic GPT2

4. Fine-tuned medical GPT2 + Fine-tuned empathic GPT2

The previous model is coupled with a GPT2 model finetuned to rewrite any text in a more empathetic way. Once the medical GPT gives a response it is fed into the empathic model to produce an empathic response which still conveys the same information.
